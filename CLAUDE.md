# CLAUDE.md - Project Guidelines for Claude Code

## Language Policy

**All code, comments, UI text, and documentation must be written in English.**

This applies to:
- Variable names, function names, class names
- Code comments and JSDoc/TSDoc
- UI strings, labels, button text, error messages
- Commit messages
- Documentation and README files
- Test descriptions

This rule applies regardless of what language the user writes in when communicating with Claude.

## Project: TIER Golf (IUP Master)

A golf training and development platform with:
- React/TypeScript frontend (apps/web)
- Fastify/Prisma backend (apps/api)
- Tailwind CSS with TIER Design System

### Design System Colors
- Navy: #0A2540 (primary)
- Gold: #C9A227 (accent)

### User Roles
- Player
- Coach
- Admin
- Parent

### Navigation Structure (V3)
5 main areas: Home, Plan, Training, Analysis, More
